# Exam Roster Lookup Tool

This is designed as a reusable tool to manage exam seating, reseating, and end-of-exam submission checks.

## How to use it

1. Upload an official seating report from the seating portal.
2. Map the required columns to create a working roster.
   - student identifiers
   - student name
   - assigned seat number
3. Update seat changes as needed (e.g., when students are moved to charging stations).
4. Paste one or multiple identifiers to look up students.

All updates are automatically saved in the session. At the end of the session, download the roster if needed, and reset the app to clear all data.

*Note: This app operates locally and does not transmit data externally. Please ensure that any data handling compiles with the institution's data governace and privacy policies.*
